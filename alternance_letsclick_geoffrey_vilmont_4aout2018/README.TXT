Bonjour et bienvenue dans ce README au sujet de ce projet.

Ce projet a été réalisé dans le cadre d'un test technique pour LetsClick pour ma candidature au poste de Developper Front-end
en alternance.

Ce projet a été réalisé en : HTML5, CSS3, Javascript, Jquery.

Le challenge pout moi a été de supprimer la scrollbar de la div contenant un long text dans le section "About".
Celui ci a été réalisé en JS et CSS3, cependant si on redimensionne la page il faudra refresh celle - ci pour avoir l'affichage correct.

Je n'ai pas encore les bonnes mécaniques d'utilisation de la structure hiérarchique Hn donc cela n'a pas été effectué parfaitement.

Mon code également n'est pas parfaitement rangé, ni 100% lisible, en effet je n'ai pas l'habitude du travail en equipe en code,
cependant j'en ai mis quelques un dans le fichier css et dans le code Jquery en esperant que ça vous aidera a mieux comprendre ma demarche.

Le site est donc responsive, mais par manque de temps je n'ai malheureusement pas vérifié si tout le code est a la norme de la W3C.

Vous verrez j'imagine mon manque d'experience et de bonnes mecaniques de developpement a la vue du code, cependant je ne souhaite que progresser.

Je vous remercie d'avoir pris la peine de lire ce fichier README,
J'ai consacré tout mon temps jusqu'a la deadline pour la realisation de ce site et j'espere l'avoir suffisamment réussi.

Cordialement,
